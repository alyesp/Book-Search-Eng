## Description 
This application allows users to search for new books form Google's Book API. Users can log in to save and remove the book they would like to purchase from their library.

## Technology 
Project was created with:
`MongoDB`
`Express.js`
`React.js`
`Node.js`
`Javascript`
`GraphQL API`
`Apollo Server`
